## Что такое Writer Studio?

Writer Studio — это **local-first** десктопное приложение для авторов художественной литературы, созданное для тех, кто хочет единую, сфокусированную среду для организации, написания и полировки рукописей — с ИИ-ассистентом, который знает вашу историю целиком.

Никаких подписок. Никакой привязки к облаку. Ваши файлы хранятся на диске в открытом формате JSON, который всегда можно прочитать и перенести.

### Зачем, если есть ChatGPT / Word / другие инструменты?

| Потребность | Типичный способ | Writer Studio |
|-------------|-----------------|---------------|
| Организация глав и сцен | Scrivener, Notion | Встроенная иерархическая структура |
| ИИ-помощь с контекстом книги | Вручную вставлять контекст в ChatGPT | ИИ читает всю рукопись автоматически |
| Проверка грамматики и стиля | Общие корректоры (не для худлита) | Нативный корректор, настроенный на литературную прозу |
| Экспорт в EPUB / DOCX | Vellum, Calibre | Экспорт одним нажатием с проверкой |
| Учёт персонажей и мира | Отдельные таблицы или вики | Карточки персонажей + графы связей |

## Возможности

### ✍️ Написание и редактирование

- Богатый текстовый редактор (TipTap) с форматированием, режимами фокуса и дзен
- Режим без отвлекающих факторов: прокрутка пишущей машинки, фокус на абзаце
- Счётчик слов, время чтения, статистика сессий в реальном времени
- Разделённый экран и многопанельный макет (до 4 панелей, произвольная вложенность)

### 📖 Структура книги

- Иерархическая организация: **Книга → Акты → Главы → Сцены**
- Перетаскивание с индикаторами статуса (*черновик / правка / финал*)
- Метаданные сцены: POV, дата/время, теги, место
- Составной редактор на уровне книги для обзора всей рукописи
- Поддержка предисловия и послесловия

### 👤 Персонажи и мир

- Карточки персонажей: внешность, характер, мотивации, отношения
- Структурные модели персонажей (3D по Эгри, арки по Вейланд)
- Визуальные графы связей
- Профили персонажей, общие между книгами серии

### 🤖 ИИ-ассистент

- **Полный контекст рукописи** — ИИ видит всю книгу, персонажей, заметки и метаданные
- **Руководство для книги (BOOK.md)** — авторские инструкции, формирующие поведение ИИ
- **Команды на естественном языке:** *«Раздели эту сцену на 3 части»*, *«Создай карточки персонажей из этой главы»*, *«Какие логические противоречия есть во 2-м акте?»*
- **Несколько ИИ-провайдеров:** Claude (Anthropic), GPT (OpenAI), Ollama (локальные модели, без интернета)
- **Свой API-ключ** — без посредников, ключ отправляется напрямую провайдеру
- **Рабочий процесс с подтверждением** — вы подтверждаете каждое структурное действие ИИ
- **ИИ-автодополнение** — ненавязчивые подсказки-призраки при написании
- **Рецензия сцены ИИ** — редакторская обратная связь с полным контекстом книги

### 🔍 Корректура

- Нативная проверка грамматики и стиля, созданная для художественной прозы
- Проверка орфографии с предложениями исправлений
- Правила с учётом части речи, понимающие диалоги, нарратив и литературные конструкции
- Подчёркивание ошибок и исправление в один клик

### 🔎 Навигация

- **Быстрый переход** (`Cmd/Ctrl+P`) — мгновенный нечёткий поиск по всей книге
- **Палитра команд** (`Cmd/Ctrl+Shift+P`) — доступ ко всем действиям в стиле IDE
- Полнотекстовый и regex-поиск по всем сценам

### 📦 Импорт и экспорт

### 🔒 Безопасность данных

- Автосохранение каждые 30 секунд с восстановлением после сбоев
- Автоматические почасовые резервные копии с настраиваемым сроком хранения
- **Открытый формат хранения** — обычные JSON-файлы, без привязки к вендору
- История версий рукописи на основе Git

- Интерфейс на русском и английском языках

## Установка

### macOS

1. Скачайте `.tar.gz` из [последнего релиза](https://github.com/RALIST/writer-studio-releases/releases/latest)
2. Распакуйте и перетащите **Writer Studio** в папку «Программы»
3. При первом запуске: правая кнопка → **Открыть**, или перейдите в **Системные настройки → Конфиденциальность и безопасность** → **Открыть всё равно**

### Windows

1. Скачайте установщик `.exe` из [последнего релиза](https://github.com/RALIST/writer-studio-releases/releases/latest)
2. Запустите установщик — если появится предупреждение SmartScreen, нажмите **Подробнее → Выполнить всё равно**

### Linux

**AppImage:**
```bash
chmod +x Writer-Studio-*.AppImage
./Writer-Studio-*.AppImage
```

**Debian/Ubuntu:**
```bash
sudo dpkg -i writer-studio_*.deb
```

## Конфиденциальность и ИИ-ключи

Writer Studio **создан с приоритетом локальности**:

- Все ваши тексты хранятся на вашем устройстве — аккаунт не нужен, синхронизации с сервером нет
- ИИ-функции **опциональны** — приложение полностью работает без ИИ-ключа
- При использовании ИИ ваш API-ключ отправляется напрямую выбранному провайдеру (Anthropic, OpenAI или ваш локальный Ollama) — Writer Studio его не видит
- Телеметрия (отчёты об ошибках) отключена по умолчанию и управляется в Настройках
